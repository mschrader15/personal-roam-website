OBD2 in the 2021 Mercedes GLE

  • As part of my Research at the University of Alabama, we are using a Mercedes-Bency GLE350 (V167) to test CV2X and traffic light optimization.

  • We intend to log simple vehicle information via the OBD2 CAN bus, and so planned on using a CANable and open-source python libraries (cantools & python-can) to write a simple logging script.

  • Lessons Learned

    • This script worked great on a 2015 VW Jetta, but moving to the V167, we have had no luck. It simply didn't respond to the standard 0x7DF OBD2 message. I saw lots of bus traffic, but none that correspond to the OBD2 standard message IDs. I was stumped.

    • Then I stumbled on this forum question about 29 bit ID's. I remembered seeing some extended CAN ID's on the bus, and decided to give it a go.

    • In extended CAN, the OBD2 request message id is 0x18DB33F1, which works!
